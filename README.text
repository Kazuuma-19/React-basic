【モダンJavaScriptとは？】
■仮想DOMとは？
・従来は直接DOMを操作
・まずは仮想DOMを操作し、差を直接DOMを操作

■パッケージマネージャーとは？
・ファイルを分割して読み込みようにする
・加えて読み込み順の操作を行ってくれる

■ECMAScriptとは？
・JavaScriptの標準規格
・ES6で大きな変更（let, const, アロー関数など）

■モジュールバンドラー、トランスパイラ
＜モジュールバンドラー＞
・webpack
・ファイルを分割し開発、その後ファイルをまとめてくれる

＜トラスパイラ＞
・babel
・新しいjs記法を古いものに変更してくれる

■SPA
＜従来＞
・その都度みたいHTMLを返却する

＜現在＞
・データのみ取得し、JavaScriptでDOMを操作。HTMLの内容を書き換える
・HTML1枚で良い
